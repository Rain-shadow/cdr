#### 如何运行？
##### 你可以采用一下指令安装第三方依赖
`pip install -r requirements.txt -i https://mirrors.aliyun.com/pypi/simple/`
##### 或者使用以下指令
`pip install Pillow qrcode requests aiohttp -i https://mirrors.aliyun.com/pypi/simple/`
##### 在确保你已经安装好相关第三方库后，使用`python main.py`执行
